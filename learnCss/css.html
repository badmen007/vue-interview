    # CSS常见技巧

    # 1.设置input的placeholder样式

    • 1.设置input占位符的样式
    
input::-webkit-input-placeholder {    /* Chrome/Opera/Safari */
    color: red;
}
input::-moz-placeholder { /* Firefox 19+ */  
    color: red;
}
input:-ms-input-placeholder { /* IE 10+ */
    color: red;
}
input:-moz-placeholder { /* Firefox 18- */
    color: red;
}

<input type="text" placeholder="请设置用户名">
    • 2.设置input聚焦时的样式
    input:focus{
    background-color: green;
}
    • 3.取消input的边框
    border: none;
outline: none;

    # 2.单行或者多行文本超出省略

    • 1.单行文本出现省略号
    width: 200px;
white-space: nowrap; /* 文本不会换行 在同一行显示 直到遇到<br/> */
text-overflow: ellipsis; /* 显示省略符号来代表被修剪的文本 */
overflow:hidden;  /* 超出的部分就不显示了 */
    • 2.多行文本出现省略号
    display:-webkit-box;
-webkit-line-clamp: 1; /* 几行 */
-webkit-box-orient: vertical;
overflow: hidden;

    # 3.负边距的使用技巧

    • 做为负时，就是左移，右为负时，是左拉。上下与左右类似
    *{
    margin:0;
    padding:0;
}
.wrap{
    /* 利用负值技巧进行整体移动 */
    margin-left:-6px;
}
.item{
    float:left;
    width: 20%;
    height: 300px;
    border-left:6px solid #fff;
    box-sizing: border-box;
}
<div class="wrap">
    <div class="item" style="background-color: red;"></div>
    <div class="item" style="background-color: green;"></div>
    <div class="item" style="background-color: yellow;"></div>
    <div class="item" style="background-color: pink;"></div>
    <div class="item" style="background-color: green;"></div>
</div>

    # 4.定位同时设置方位的情况

    • 绝对定位和固定定位时,同时设置left和right等同于隐式设置了宽度
    span{
    border:1px solid red;
    position: absolute;
    left:0;
    right:0;
    
     /* 等同于设置  width:100%;display:block */
}

<span>1</span>

    # 6.outline属性的使用技巧

    • outline不计算大小 border计算大小
    * {
    padding: 0;
    margin: 0;
}

ul {
    list-style: none;
    width: 600px;
    margin: auto;
}

li {
    padding: 10px;
    border: 10px solid pink;
    outline-offset: -10px;
}
li+li{
    margin-top:-10px;
}
li:hover{
    /* border:10px solid gold; */
    outline:10px solid gold;
}
<ul>
    <li>1</li>

    /* 
        1.transfrom:translate(x, y) translateX() translateY()
        2.transform: scale(x, y) scaleX() scaleY()
        3.transform: rotate(deg)  角度 rotateX() rotateY() rotateZ() + transform-origin: 固定基点；
     */
     *{padding:0;margin:0}
     body{
         background: black;
     }

         /* 为了鼠标移入移出都有过渡动画的效果 transition加在没有hover元素本身上 
            + 动画曲线 transition-timing-function
            + ease
            + ease-in
            + ease-out
            + ease-in-out
            + liner
         
         */
         transition: transform 1s ease; 
        }
     .right li:hover{ 
         /* transition 一起使用 过渡 */
        transform: translate(0, -5px);
     }

     }

         background-color: rgb(255,255,255,.5);

         transform: rotate(0deg);

        transition 过渡动画
        1. transition-property 过渡动画属性名称(css属性)
        2. transition-duration 过渡动画持续时间(时间越长 动画越慢 反之越快)
        3. transition-timing-function 过渡动画动画 运动曲线(ease匀速)
        4. transition-delay 过渡动画动画延迟时间 (一般不写)

        符合属性:transition-property transition-duration transition-timint-function transition-delay
        transition: width 1s ease 2s

      */
</style>

    animation

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>animation动画和交互</title>
</head>
<style>
    /* 
        animation-name:动画名称
        animation-duration 动画持续的时间 
        animation-timing-function 动画曲线
        animation-delay 动画延迟时间
        animation-itration-count: infinite 运动次数
        animation-direction： alternate 动画的交替运动 
        animation-play-state: paused /running 动画的运动、暂停
        animation-fill-mode: forwards; 运动完成之后 forwards 停止在100%的位置 backwards 停在原点位置
     */
   .red{
        width:200px; 
        height:200px;
        background-color: red;
        border-radius:50%;
        animation-name:RedBox;
        animation-duration:1s;
        animation-timing-function: linear;  /*线型曲线更加的平滑 */
        animation-iteration-count:infinite;
        animation-direction: alternate;
        animation-play-state: alternate;
        animation-fill-mode: forwards;


    /* animation: RedBox 1s linear alternate backwards; */
   }


   /* 
    transform:
        translate
        scale
        rotate
   如果三个属性同时存在的话
   transform: scale() rotate() translate();
    */
   @keyframes RedBox {
    0%{transform: translate(0,0) rotate(0deg)}
    50%{transform: translate(400px,0)}
    100%{transform: translate(800px, 0) rotate(90deg);background-color: royalblue}
   }
   /* css3动画库 animate.min.css */
</style>
<body>
    <!-- 
        写动画分成两步:
        1.通过@Keyframes 自定义动画名称
        2.通过animation属性去调用定义好的名称
        @keyframes 动画名称 {
            0%  - from
            .....
            100% - to
        }
     -->
     <div class="red"></div>
</body>
</html>
